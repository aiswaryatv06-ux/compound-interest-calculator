# Compound Interest Calculator

## Description
This is a simple Python program that calculates compound interest based on principal amount, interest rate, and time period.

The program takes user input and calculates the final amount using the compound interest formula.

# Features
- Takes principal amount as input
- Takes interest rate and time period
- Calculates compound interest
- Displays final amount

# Technologies Used
- Python

# How to Run
1. Download the file
2. Open terminal or command prompt
3. Run:
   python compound_interest.py

# Formula Used
A = P(1 + r/n)^(nt)

Where:
- P = Principal amount
- r = Interest rate
- n = Number of times interest applied per year
- t = Time in years
- A = Final amount
